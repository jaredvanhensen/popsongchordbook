Option 1: Nebula Dark Glass (Ultra-Premium Glassmorphism)

A beautiful combination of highly translucent, dark glass with a high blur index and light gradient reflecting at the top-left edge. This provides excellent readability, keeps transparency, and blocks bright colors underneath from washing out the container.

.lyrics-hud {
    background: linear-gradient(135deg, rgba(255, 255, 255, 0.08) 0%, rgba(10, 10, 10, 0.55) 100%);
    backdrop-filter: blur(24px) contrast(105%);
    border: 1px solid rgba(255, 255, 255, 0.22);
    box-shadow: 
        inset 0 1px 1px rgba(255, 255, 255, 0.3), 
        0 12px 40px rgba(0, 0, 0, 0.45);
}
