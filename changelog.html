Changelog

Summary of changes and updates for Pop Song Chord Book

v2.118 February 24, 2026
Default Sorting: The song list now defaults to sorting by Song Title (ascending) automatically when the app loads.
Visible Scrollbar: Re-enabled the scrollbar for the song list table with a custom premium design (sleek indigo thumb).
v2.117 February 24, 2026
Timestamp Optimization: All chord timestamps in the database and defaults are now rounded to 3 decimal places (milliseconds). This significantly reduces data size.
Version Update: Updated index.html, songlist.html, scrolling_chords.html, and js/app.js to reflect the new version number.
Cache Busting: Updated all script and CSS query strings to ?v=2.117 to ensure all users receive the latest code immediately.
v2.116 February 23, 2026
Audio Engine: Perfect audio engine revert to v2.098 while preserving chord parsing logic for maximum reliability.
v2.108 - v2.115 February 23, 2026
Profile Levels: Added a premium Level Badge system to the Profile button based on practice consistency.
Practice Stats: Added "Top 10 Practiced Songs" table and total session counter to the profile modal.
Audio Samples: Integrated Premium Grand piano samples (Yamaha C5) and Studio Reverb options.
v2.104 - v2.106 February 22, 2026
Grid Precision: Increased Bar adjustment precision to 0.25 beats (16th notes) and refined snap-to-grid to 16 subdivisions.
Virtual Piano: Added a compact virtual piano interface in EDIT mode for quick reference.
v2.093 - v2.098 February 22, 2026
Redo Function: Implemented Redo functionality (Ctrl+Y) on the chord timeline.
Timeline Fixes: Fixed 'jumping' chords by removing laggy transform transitions and conflicting hover transforms.
Drag & Drop: Implemented drag-and-drop chords from the toolbar directly to the timeline.
v2.080 - v2.091 February 20, 2026
Mobile UI: Extensive refinements to the footer layout, icon alignment, and badge sizing for mobile devices.
BPM & Key: Unified BPM and Key fields with interactive clickable badges and fixed sizing.
v2.043 - v2.052 February 19, 2026
Tablet Optimization: Fixed layout issues, timing lags, and scrolling discrepancies specifically for iPad and tablets.
Lyrics HUD: Optimized the mobile portrait Lyrics HUD for a cleaner 2-line display.
v2.030 - v2.042 February 17-19, 2026
Edit Mode: Introduced the Edit Mode toggle to declutter the toolbar for general users.
Stabilization: Grouped adjustment buttons on a stable 3rd row for mobile and stabilized the toolbar.
v2.015 - v2.023 February 13-17, 2026
Shifting Controls: Added bulk chord shifting controls to the timeline.
Lyrics HUD: Added the Lyrics HUD Teleprompter and LRC file parser.
Custom Modals: Replaced browser native confirm/alert boxes with professional custom designed modals.
